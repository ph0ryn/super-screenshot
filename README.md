# SuperScreenshot

SuperScreenshot is a Chrome extension for copying screenshots from several
videos and livestreams frames to the clipboard.

## Installation

1. Download .zip from latest release and unzip it.
2. Open chrome extension settings and click `Load unpacked`.
3. Select the unzipped.

## Usage

1. Press `p` outside text inputs.

## Supported

- Twitch
- YouTube
- Instagram live
